Code and Culture:

A practice-led visual comparison of how generative AI platforms construct youth-coded game characters under aligned prompts and PG constraints. This page presents side-by-side/slider evidence, concise notes, and reproducible metadata.

Introduction

  • Same prompts, different platforms (EN vs CN contexts).
  • PG / non-sexualized constraints applied across all images.
  • Focus dimensions: coverage, pose, style, palette, symbols.
  • Method: prompt parity + slider comparison + participant notes.

Midjourney

Diffusion-based text-to-image via Discord. Tends toward semi-realistic / heroic rendering, dramatic lighting and material detail. Good parameter control (AR, seed, stylize) for reproducibility.

  • Style: realism/semi-realism, functional gear, environmental narrative.
  • Default pose often neutral/controlled; higher coverage.
  • Record: prompt, negatives, AR, seed, time.

Wenxin Yige

Chinese text-to-image platform with strong support for CN prompts and style tags. Frequently outputs anime/illustration, cleaner foregrounds, higher saturation.

  • Style: anime/illustration, kawaii/symbolic tendencies.
  • Default pose often icon/turn-body; coverage can be lower by default.
  • Record: prompt CN/EN, style tag, time.

Keyword

Prompts follow CN/EN parity with negative constraints (PG / non-sexualized). Metadata template includes: platform, prompt EN/CN, style/params/seed, time, negatives, notes.

Analysis

Patterns: Midjourney tends toward realism/semi-realism, higher coverage and neutral stance with environmental narrative; Wenxin Yige tends toward anime/illustration, icon-like poses, higher saturation and cleaner foregrounds.

Implications: Under the same “youth-coded + PG” semantics, platform defaults still shape coverage, pose and gaze. For child/adolescent contexts, set conservative defaults (modest clothing, neutral stance) and document negatives.
